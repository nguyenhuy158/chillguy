# 🎧 chillguy

A terminal-based YouTube music player designed for simplicity and style. Stream your favorite tunes directly in your terminal with a "chill" aesthetic.

## ✨ Features

- **Search:** Find any track or playlist on YouTube.
- **Interactive Player:** Beautiful UI with progress bars and status updates.
- **Keyboard Controls:** 
  - `space`: Play/Pause
  - `->` / `<-`: Seek 5s
  - `+` / `-`: Volume up/down
  - `q`: Quit
- **System Doctor:** Built-in dependency check.

## 🚀 Installation

### Prerequisites

You need `mpv` installed on your system for audio playback.

- **macOS:** `brew install mpv`
- **Linux:** `sudo apt install mpv`
- **Windows:** Download from [mpv.io](https://mpv.io/)

### Install chillguy

```bash
pip install chillguy
```

## 🛠 Usage

Check if you have everything installed:
```bash
chillguy doctor
```

Play music:
```bash
chillguy play "lofi hip hop radio"
```
